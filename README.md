# LibanCom

LibanCom is a digital identity platform. Create shareable profiles with lead capture, analytics, and customizable themes.

## Features

- **Digital Profiles**: Customizable profiles shared via link or QR code
- **Lead Capture**: Track networking interactions and export leads to CRM
- **Teams/Organizations**: Enterprise features for managing employee digital cards
- **Analytics**: Track profile views and engagement metrics

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
